🌈 Sumate al Discord de Transistemas 🏳️‍⚧️

Seguí estos pasos para vincular tu cuenta y acceder a los canales exclusivos de estudiantes.

1
Descargar Discord
Si todavía no tenés Discord instalado descargalo desde la página oficial.
Descargar Discord
2
Unirte al servidor de Discord de Transistemas
Iniciá sesión en Discord y sumate al servidor oficial de nuestra comunidad.
Unirme al servidor de Transistemas
3
Conectar Discord a la API de Transistemas
La API se encargará de obtener tu Discord ID usando OAuth.
4
Acceder a canales exclusivos de Estudiantes
Una vez vinculada tu cuenta, podés solicitar acceso a los canales de estudiantes.
